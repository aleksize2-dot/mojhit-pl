# Фундамент проекта (FP) — mojhit.pl

Этот документ — выжимка нашей архитектуры, бизнес-логики и видения продукта. Шпаргалка, чтобы в любой момент быстро вспомнить, как всё устроено и куда мы движемся.

## 1. Архитектура (Vibe Coding Stack)
- **Frontend:** React + TailwindCSS (Vercel/Pages). Строгий Mobile-first, sticky bottom nav (прилипающая нижняя навигация), жирный футер для SEO, Consent Mode v2 (куки-баннер).
- **Backend:** Node.js/Express, развернутый на VPS.
- **База данных:** Supabase (PostgreSQL).
- **Авторизация:** Clerk (Социальные сети, Email).
- **Хранилище:** Cloudflare R2 (оптимально для хранения MP3/Аудио).
- **Платежи:** Stripe (с критически важной поддержкой BLIK для рынка Польши).

## 2. Бизнес-модель и Экономика
- **Двойная экономика:** 
  - **Монеты 🪙** (Платная валюта: 1 Монета = 1 Генерация / 2 трека).
  - **Ноты 🎵** (Бесплатная валюта/Фарм: 10 Нот = 1 Монета). Добываются через Daily Login (ежедневный вход), мини-игры во время ожидания генерации и за лайки от авторизованных пользователей.
- **Виральная петля (Social Proof):** Шаринг сгенерированных треков через красивые OpenGraph карточки. Гости должны зарегистрироваться, чтобы поставить лайк (а лайк приносит автору трека Ноты).
- **Лид-магнит и Эффект приманки (Decoy Pricing):** 
  - Пользователи могут генерировать треки без регистрации (Shadow Accounts с привязкой по `guest_session_id`).
  - Для неавторизованных юзеров воспроизведение обрывается на 40-й секунде (плавный fade out).
  - Пакеты: 15 PLN (просто скачать 2 трека — невыгодно), 49 PLN (2 трека + 10 Монет + авто-создание аккаунта — оптимально), 99 PLN (VIP + 30 Монет).
  - **Магия вебхуков:** При оплате через Stripe вебхук автоматически создает пользователя в Clerk/Supabase, разблокирует треки, начисляет монеты и отправляет на почту полные MP3 вместе с временным паролем (через Resend). Так мы получаем лид и подписку на рассылку.

## 3. Админ-панель (admin.mojhit.pl)
- Отдельное SPA-приложение для CRM, Дашбордов (DAU, Revenue, Топ Продюсеров), Модерации треков, управления Промокодами, Feature Flags и мониторинга API-балансов (OpenRouter, Kie/Suno).
- **Управление Пользователями:** CRM с возможностью ручного начисления/списания Монет и Нот, смена статуса аккаунтов (Бан, Shadow).
- **Биржа Талантов (Управление Продюсерами):** Модуль для добавления, редактирования и удаления ИИ-Продюсеров. 
  - **CRUD операции:** Добавление нового продюсера (загрузка аватарки, настройка System Prompt, выбор стиля, модели).
  - **Редактирование:** Изменение параметров существующих продюсеров "на лету" без деплоя.
  - **Монетизация:** Установка цены в Монетах за временную "аренду" продюсера.
  - **Вкл/Выкл:** Активация или деактивация продюсеров (например, сезонные персонажи).
- **Модерация и Треки:** Лента всех генераций с плеером, возможностью блокировки пользователей и удаления нарушающего правила контента.

## 4. ИИ-Агенты (Core Agents)
- **Hit-Assistant:** Умный ИИ-чатбот техподдержки, плавающий поверх интерфейса. Если не справляется — эскалирует тикет в Админ-панель живым людям.
