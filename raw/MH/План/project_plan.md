# Проект: mojhit.pl (AI Music Generator for Poland)

## 📌 Описание
Генератор AI-музыки с фокусом на польский рынок. Пользователи могут создавать треки, используя "вибрации" и шаблоны (например, Wedding -> Polish Disco Polo).

## 🏗 Архитектура (Vibe Coding Stack)
- **Frontend:** React + TailwindCSS (Vercel/Pages). Mobile-first, sticky bottom nav, fat footer для SEO, Consent Mode v2 (Cookie Banner).
- **Backend:** Node.js/Express, деплой на VPS.
- **Database:** Supabase (PostgreSQL).
- **Auth:** Clerk (Social Login, Email).
- **Storage:** Cloudflare R2 (для MP3/Audio).
- **Payments:** Stripe (критична поддержка BLIK для Польши).

## 💰 Бизнес-модель и Экономика
- **Двойная экономика:** 
  - Монеты 🪙 (Платные, 1 Монета = 1 Генерация/2 Трека).
  - Ноты 🎵 (Бесплатные/Фармятся, 10 Нот = 1 Монета). Добываются через Daily Login, мини-игры во время загрузки и лайки от авторизованных пользователей.
- **Viral Loop (Social Proof):** Шеринг треков через OpenGraph. Гости должны зарегистрироваться, чтобы лайкнуть трек (что дает автору Ноты).
- **Лид-магнит и Decoy Pricing:** 
  - Пользователи могут генерировать без авторизации (Shadow Accounts через `guest_session_id`).
  - Воспроизведение лочится на 40 секундах (fade out).
  - Уловки с ценой (Decoy Pricing): 15 PLN (просто скачать 2 трека), 49 PLN (2 трека + 10 Монет/создание аккаунта), 99 PLN (VIP + 30 Монет).
  - Магия вебхуков: Оплата Stripe создает юзера в Clerk/Supabase, анлокает треки, начисляет монеты и отправляет фулл MP3 + временный пароль через Resend (Lead Magnet + Newsletter opt-in).

## 🎛 Админ-панель (admin.mojhit.pl)
- Отдельное SPA-приложение для CRM, Дашбордов (DAU, Revenue, Топ Продюсеров), Модерации треков, управления Промокодами, Feature Flags и мониторинга API-балансов (OpenRouter, Kie/Suno).
- **Управление Пользователями:** CRM с возможностью ручного начисления/списания Монет и Нот, смена статуса аккаунтов (Бан, Shadow).
- **Биржа Талантов (Управление Продюсерами):** Модуль для добавления, редактирования и удаления ИИ-Продюсеров. 
  - **CRUD операции:** Добавление нового продюсера (загрузка аватарки, настройка System Prompt, выбор стиля, модели).
  - **Редактирование:** Изменение параметров существующих продюсеров "на лету" без деплоя.
  - **Монетизация:** Установка цены в Монетах за временную "аренду" продюсера.
  - **Вкл/Выкл:** Активация или деактивация продюсеров (например, сезонные персонажи).
- **Модерация и Треки:** Лента всех генераций с плеером, возможностью блокировки пользователей и удаления нарушающего правила контента.

## 🤖 Core Agents
- **Hit-Assistant:** AI-чатбот поддержки, плавающий в UI, с возможностью эскалации в админку.
