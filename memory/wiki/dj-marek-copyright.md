# DJ Marek - Авторские права и владение треками

## Контекст
**DJ Marek** — это AI-композитор в приложении MojHit.pl, который генерирует промпты для Suno AI на основе описания пользователя.

## Цепочка создания трека
1. **Пользователь** описывает идею трека (текст промпта)
2. **DJ Marek (AI)** преобразует описание в структурированный промпт для Suno
3. **Suno AI** генерирует аудиотрек (музыка + вокал)
4. **Трек** сохраняется в базе MojHit.pl и становится доступен для скачивания

## Вопрос авторских прав

### Права пользователя
- Пользователь является **автором текстового описания** (промпта)
- Сгенерированный трек создаётся **на основе промпта пользователя**
- **MojHit.pl** не претендует на владение сгенерированными треками
- Пользователь может **свободно использовать** треки для личных и коммерческих целей

### Права Suno AI
- **Suno AI** имеет собственные условия использования (Terms of Service)
- Suno может претендовать на **некоторые права** на сгенерированный контент
- Пользователь должен ознакомиться с [Suno Terms of Service](https://www.suno.ai/terms)

### Права MojHit.pl
- **MojHit.pl** предоставляет платформу для генерации треков
- Платформа **не заявляет авторских прав** на сгенерированный контент
- Треки хранятся в базе MojHit.pl для удобства пользователя
- Пользователь может удалить свои треки в любой момент

## Необходимые уведомления
В интерфейсе MojHit.pl следует добавить уведомление:

> **Уведомление об авторских правах**
> 
> Треки, сгенерированные через Suno AI, являются собственностью пользователя, создавшего промпт. Однако Suno AI может иметь собственные условия использования сгенерированного контента. Ознакомьтесь с [Suno Terms of Service](https://www.suno.ai/terms) для получения подробной информации.
> 
> MojHit.pl не претендует на права владения сгенерированными треками.

## Места для размещения уведомления
1. **Страница генерации трека** (перед кнопкой "Generate")
2. **Страница деталей трека** (в разделе информации)
3. **Политика приватности** (раздел "Генерация треков AI")
4. **Regulamin** (условия использования)

## Правовые риски
- **Риск:** Suno AI может изменить условия и потребовать лицензионных отчислений
- **Митигация:** Явное уведомление пользователя о сторонних условиях
- **Риск:** Пользователь может использовать треки для незаконного контента
- **Митигация:** Regulamin должен запрещать генерацию треков с незаконным содержанием
