## 📝 Расшифровка колонок

### Статусы:
- 🔵 **NEW** — найдено, ещё не писал
- 🟡 **CONTACTED** — написал, жду ответа
- 🟠 **NEGOTIATING** — обсуждаем цену/формат
- 🟢 **ACTIVE** — договорились, пост опубликован
- 🔴 **REJECTED** — отказались / дорого
- ⚫ **BANNED** — забанил / не отвечает
- 🔄 **REPEAT** — отработал, можно через месяц повторить

### Тип канала:
- **AI** — AI art / AI tools / AI news
- **WAIFU** — AI waifu / AI girlfriend / character
- **CRYPTO** — крипта / заработок
- **TECH** — технологии / стартапы
- **CREATIVE** — дизайн / art / фото

### Формат публикации:
- **POST** — пост в ленту
- **PIN** — пост + закреп
- **DIGEST** — упоминание в дайджесте
- **STORY** — сторис
- **CHAT** — сообщение в чате
- **CROSS** — обмен постами

### Результат (30 дн):
- 🤯 **FIRE** — >100 переходов, >5 платящих
- 👍 **GOOD** — 30-100 переходов, 2-5 платящих
- 😐 **MEH** — <30 переходов, <2 платящих
- 👎 **FAIL** — 0 платящих

## ⚡ Быстрые действия

### Перед первым контактом:
- [ ] Написать готовый текст поста для админа
- [ ] Сделать картинку/видео-пример генерации
- [ ] Создать промокоды: INFLUENCER20, CHANNELNAME (каждому каналу уникальный)
- [ ] В боте — аналитика переходов по промокодам

### После публикации:
- [ ] Отследить количество переходов в первые 24ч
- [ ] Посчитать покупки Stars через промокод
- [ ] Записать результат в CRM
- [ ] Через 7 дней — написать "Привет, результаты отличные, можем повторить?"
