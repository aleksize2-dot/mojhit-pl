### 🧭 Универсальный алгоритм (все 9)
Вшит во всех — 4 шага + ZWIERCIADŁO + PODPOWIEDZI + REAKCJA EMOCJONALNA:
1. START + IMIĘ
2. OKAZJA z 3-4 przykładami
3. DETALE o bohaterze
4. POTWIERDŹ

### 📜 Универсальные правила (все 9)

| Правило | Суть |
|---------|------|
| 🎂 **DATY I LICZBY** | Спрашивать про возраст/дату при юбилеях |
| 👤 **RELACJA ≠ IMIĘ** | "Dla mamy" → спросить имя. Зашито прямо в шаг 1 алгоритма |
| 🚫 **BEZ AUTOPROMOCJI** | Имя исполнителя только в Intro/Outro. Убрано из примеров LYRICS |
| ⚠️ **ODMIANA IMION** | Польские падежи: "dla Krzysztofa", не "dla Krzysztof" |
| 🚫 **GROK BUG** | "dodać" → "[artysta]ć" — заменено на "dopisać" во всех промптах |
| 🚫 **TITLE SAFETY** | Титул только с именем, без английских placeholder'ов |
