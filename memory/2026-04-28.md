### 🎬 Video Generation — Complete Rewrite & Fixes

**Decision (2026-04-28):** Video generation available for ALL subscription tiers, not just PRO.
- Free → watermark (`mojhit.pl`)

**Bugs fixed (8 issues resolved):**
1. `audioId` mismatch — Kie.ai API expects Suno audio ID (UUID), not Kie `task_id`. 3-level fallback: (a) `suno_id:xxx` from variant tags, (b) base64-decode from `musicfile.kie.ai/` URL, (c) KIE record-info API call, (d) task_id as fallback
2. Webhook code check — Kie callback sends `code: 0` for success, not `200`. Fixed to accept both.
3. Video status parser — Kie returns `successFlag: "SUCCESS"` (not `status`) and `response.videoUrl` (nested). Fixed.
4. Audio CORS — `musicfile.kie.ai` CDN doesn't set CORS headers. Added `/api/proxy/audio` endpoint.
5. Same video for V1/V2 — Both variants shared `kie_tasks.id`. Fixed: key video by `kie_track_variants.id` per variant.
6. Duplicate video tasks — Added dedup check before generating.
7. Missing variant 2 tracks — `audio_url` was null in callback, `stream_audio_url` had the URL. Fixed fallback.
8. Audio on track page — Tracks with `processing` status blocked video. Relaxed check.

**New features:**
- Auto-video generation: Webhook triggers video for first variant after track completion
